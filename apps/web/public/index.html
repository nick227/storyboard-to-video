Workflow

One project from source text to exported media.

01

Import the script

Paste a story or screenplay. Gemini or OpenAI converts it into editable scene beats, image prompts, and narration text.

02

Generate scene images

Select a built-in style or upload character and environment references. Generate each frame with Gemini, OpenAI, or Dezgo.

03

Create narration

Use Piper or ElevenLabs text-to-speech, or record a sample and clone your narrator voice with the Spark-TTS service.

04

Add motion and export

Convert scene images into clips with LTX-Video. Preview clips with narration, reorder scenes, and download the project assets.
